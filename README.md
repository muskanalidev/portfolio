# Muskan Ali - Personal Portfolio

A modern, interactive personal portfolio website showcasing my work in Analytics, Business Intelligence, AI, and Product Strategy. Built with React, TailwindCSS, and Vite.

## 🚀 About Me

Hi, Muskan here.

I'm stepping into my final year and currently interning at ISRO, where I analyze 50 years of satellite-based atmospheric ozone data to identify long-term trends and build predictive models to forecast ozone layer variations. My work focuses on using data and analytics to turn ideas into outcomes that drive smarter business and product decisions.

**Skills:** SQL, Power BI, Data Analytics, Product Strategy, Business Intelligence, Python

Outside of work, I’m chronically making content, playing football, chasing beach sunsets, and filling my camera roll with way too many photos.

## 💻 Tech Stack

- **Framework:** React
- **Styling:** TailwindCSS
- **Build Tool:** Vite
- **Icons:** Lucide Icons
- **Effects:** Custom WebGL and Canvas animations

## 🌟 Features
- **Interactive Background:** A stunning Liquid Ether WebGL background.
- **About:** Background, skills, and personal interests.
- **Experience:** Professional timeline, including my internship at ISRO.
- **Projects:** Featured projects highlighting data analysis and predictive modelling.
- **Achievements:** Key milestones and accomplishments.
- **Beyond Work:** A glimpse into my life outside of analytics.

## 🛠️ Running Locally

1. Clone the repository
```bash
git clone https://github.com/muskanalidev/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```
